使用 Python 爬取 W 站图片

发表于 2019-05-08 | 更新于 2019-05-11

最近读到一篇关于 python 爬虫的博客，对 python 爬虫产生了兴趣，所以找了个网站来爬取图片练练手。

首先用 chrome 打开网页 www.wtfuck.net。

找到切换页面的标签Older posts，在此标签上右键单击，然后选择检查，找到Older posts对应的连接为 https://wtfuck.net/page/2/。

由此可以知道该网址的页面切换逻辑为从 https://wtfuck.net/page/1/ —>https://wtfuck.net/page/2/ —>https://wtfuck.net/page/n/，改变链接中最后一位数字即可实现页面的切换。

我们打开一个Older posts看一看。

可见一个 page 中有多个主题（theme），我们找到一个主题的标题，同样的方法，在上面右键然后选择检查，就可以找到该主题对应的链接。

然后我们打开一个主题（theme）。

然后找到一张图片右键—>检查，即可找到该图片的链接。

有了该链接我们就可以将图片保存下来了。
