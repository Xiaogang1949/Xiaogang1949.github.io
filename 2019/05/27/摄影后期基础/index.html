摄影后期基础操作

发表于 2019-05-27

接片、HDR、景深合成、堆栈

接片

准备

素材中的照片要有 1/4 以上的重叠部分

操作方法

Photoshop中，选择文件–>自动–>Photomerge，点击浏览加载文件，点击确定，Photoshop会自动进行合并，然后选择拼合图层，再裁剪掉不规则区域就完成了接片。

HDR合成

准备

用三脚架固定机位拍摄，然后用包围曝光的方法连续拍多张不同亮度的照片

操作方法

Photoshop中，选择文件–>自动–>合并到HDR，点击浏览选取照片，载入之后，在对话框中勾选[尝试自动对齐源图像]，然后点击确定，在弹出的对话框中调节相关的参数，点击确定即完成了合成。

景深合成

准备

用三脚架固定机位，使用大光圈、M 档拍摄几张构图相同曝光参数相同但对焦点不同的照片。

操作方法

Photoshop中，选择文件–>脚本–>将文件载入堆栈，在对话框中勾选[尝试自动对齐源图像]，点击确定，然后同时选中导入的几个图层，选择编辑–>自动混合图层，选择堆叠图像，点击确定，最后将图层合并。

堆栈

准备

使用定时快门线设置延时启动时间、曝光时间、间隔时间和拍摄数量几个参数，拍摄一组素材。

操作方法

Photoshop中，选择文件–>脚本–>将文件载入堆栈，在对话框中勾选[尝试自动对齐源图像]和[载入图层后创建智能对象]，选择图层–>智能对象–>堆栈模式–>平均值，裁剪画面，合并图层，使用CameraRaw滤镜调节色彩。
